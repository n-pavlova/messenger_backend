# messenger_backend
## Scripts to get started

* To start a server part, please run for Windows:

```
pip install virtualenv
virtualenv messenger_env
messenger_env\Scripts\activate
pip install -r requirements.txt
python3 main.py
```

* For Linux: 
```
pip install virtualenv
python3 -m venv ./venv
source ./venv/bin/activate
pip install -r requirements.txt 
python3 main.py
```
